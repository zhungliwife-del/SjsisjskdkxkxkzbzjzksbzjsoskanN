# DK University — SillyTavern UI Themes 🎓

Three matching ultra-modern glass themes for DKU RPG chats. Import via **User Settings → UI Theme → Import**.

| File | Vibe |
|------|------|
| `DKU-Old-Money-Afterdark.json` | 🌑 dark · champagne gold + violet · main theme |
| `DKU-Daylight-TrustFund.json` | ☀️ light · cream glass + espresso/indigo |
| `DKU-Spotted-GossipNeon.json` | 👁 bonus · gossip-forum neon pink on black |

Shared DNA:
- Avatar strip (char & user) sized like **Midnight Ocean** — full-width 350px with smooth bottom fade
- Input pill sized like **Bayou** with an animated shimmer catchphrase
- Top/bottom bar icons swapped to human-readable glyphs (**Heartbeat** technique)
- Wine & Cherry Pie-style labeled stat chips: PLAY № / STAT LINE / SHOT CLOCK (sports-renamed per theme)
- Animated infoboards (tables, blockquotes, spoilers), themed visible toasts, mobile-friendly (@1000px / @480px)

For token count, timer and message № chips: enable **Message Timer**, **Message IDs** and **Token Count** in User Settings (bundled in the theme JSON already).
