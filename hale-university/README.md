# 🏛 HALE UNIVERSITY — SillyTavern UI Themes

Три темы по вселенной [Hale University](https://haleuniversity.netlify.app/) — элитный универ в Нью-Йорке. Ультра-модерн, стекло (glassmorphism), phone-like, никакой классики.

## Темы

| Файл | Тема | Вайб |
|------|------|------|
| `HALE-ROYALTY-old-money-never-sleeps.json` | 👑 HALE ROYALTY | тёмное стекло, шампань-золото + ледяной синий, old money |
| `HALE-DAYLIGHT-trust-fund-era.json` | 🎓 HALE DAYLIGHT | светлая: матовое белое стекло, королевский синий + золото |
| `HALE-COURTSIDE-knights-after-dark.json` | 🏀 HALE COURTSIDE | неон: фиолет + циан, арена Knights в полночь |

## Установка

User Settings → UI Theme → Import (иконка импорта) → выбрать `.json`.

## Что внутри

- **Аватары** персонажа и юзера — размеры как в Midnight Ocean: на всю ширину, 350px, мягкое затухание снизу (260px на мобиле)
- **Поле ввода** — размеры как в Bayou + анимированная переливающаяся фраза-плейсхолдер
- **Иконки** верхнего бара и нижней панели заменены (как в Heartbeat) на человекочитаемые
- **Контейнер сообщения** в духе wine & cherry pie: чипы `PLAY №` (номер сообщения), `STATS` (токены), `SHOT CLOCK` (время генерации), кнопка размышлений «locker room talk», кастомные чекбоксы, свайпы `TAKE x/y`
- **Уведомления** — стеклянные тосты с эмодзи и цветными полосами, всегда видимые
- **Инфобоксы** (blockquote / details / таблицы) — стекло + анимированные градиентные полосы
- Спортивные переименования (баскетбол/футбол/хоккей), мат, мемы, каомодзи — в комплекте
- Мобильная адаптация включена

Рекомендуемые настройки уже зашиты в JSON (bubble chat, ширина 50, таймер/токены/ID включены).
